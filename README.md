### Teclonogias Usadas:
* Java 1.8.0_201
* Spring Boot 2.1.3
* Banco MySQL 8.0.15
* Flywa 3.2.1
* Apache Maven 3.3.3


# Para executar:
* Importar Projeto como maven project

* configurações do banco de dados em:
  - src/main/resources/application.properties

* Executar o projeto:
  - Abrir a classe DesafioSuperHeroisApplication e executa-la

* Acessar projeto em:
  - http://localhost:8080
